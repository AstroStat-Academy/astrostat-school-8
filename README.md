# AstroStat School 2026 — New York

This repository contains the lecture materials for the **8th AstroStatistics School**, New York 2026.

The book is published at: [astrostat-academy.github.io/astrostat-school-8](https://astrostat-academy.github.io/astrostat-school-8/)


## Instructions for contributors

- Add your folder to the repo
- Ensure the exact version of the packages you need are present in `environment.yml`
- Add the relative path to the notebook in `_toc.yml`
- Add the **static paths** to the `data` adn `images` sub-folders  in the `_config.yml`
- Add a short description of your notebook in `index.md`
